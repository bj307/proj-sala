   Imagens com CSS

O CSS foi criado na intenção de separar o conteúdo do estilo da página, e com essa organização foram incrementadas várias atribuições e propriedades de estilização, evitando também a repetição de propriedades ao longo do HTML. A manipulação pode ser feita somente com HTML, mas o ideal é utilizar o CSS para auxiliar não só na organização da página mas também porque possui mais formas de modificar a imagem de forma que corresponda as ideias do desenvolvedor.

Com CSS é possivel manipular a forma de utilização das imagens de inumeras formas, algumas delas são:

  • Arredondando as bordas da imagem
  • Alterando o tamanho da imagem
  • Imagem Responsiva
  • Centralizar imagem de forma responsiva
  • Editar imagem de fundo
  • Fade In
  • Crescer (grow) e Encolher (shrink)

  • Imagem com Fade In.
    • O efeitos Fade in é um dos efeitos mais comuns entre os programadores. É uma ótima maneira de enfatizar a funcionalidade ou chamar a atenção para uma chamada à ação. O efeito Fade in é codificado em duas etapas: primeiro, você definir o estado inicial; Em seguida, você define a mudança, utilizando por exemplo o seletor de evento hover.
